# 🔎 서비스 소개 Introduction
#### `곰방`은 계약기간이 남은 방을 급하게 내와야 하는 사람, 단기 자취방을 구하고 싶은 사람들이 부동산을 거치지 않고 복비를 이중으로 지출하지 않고 빠르게 양도할 수 있는 서비스입니다.
#### 양도자와 양수자가 직접 연락해 결정하기 때문에 복비 지출을 줄일 수 있습니다.
#### 실시간 채팅과, 화상 통화를 통해 직접 집을 방문하지 않고 자세한 사항을 볼 수 있습니다.
